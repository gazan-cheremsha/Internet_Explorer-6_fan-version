# 🌐 Retro Internet Explorer 6 (Fan Edition 2026)

Добро пожаловать в проект легендарного браузера **Internet Explorer 6**, воссозданного с нуля!

## 🔥 Что внутри:
* 🛠 **Связка технологий:** Написан на Python (PyQt5) + скомпилирован через суровую Nuitka!
* 🕵️‍♂️ **Умный VPN:** Автоматически открывает заблокированные сайты (например, `mult.ru` для просмотра Масяни).
* 📜 **Рабочий Журнал:** Классическая ретро-панель истории Windows XP со шрифтом Tahoma.
* 🔍 **Домашняя страница:** Старый добрый Google.

## 🚀 Как запустить:
1. Зайди в раздел **Releases** справа.
2. Скачай архив `main.dist.rar`/ `main.dist.v2.rar`
3. Распакуй через WinRAR на диск `C:` и запускай `main.exe`
## Особенности
1. Нет поддержки виджетов-чатов
